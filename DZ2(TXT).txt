 1. Создать внешний репозиторий c названием TXT.

 2. Клонировать репозиторий TXT на локальный компьютер.                       // git clone https://github.com/Rodon1234/TXT.git

 3. Внутри локального TXT создать файл “new.txt”.                             //  touch new.txt

 4. Добавить файл под гит.                                                    // git add . 

 5. Закоммитить файл.                                                         // git commit -m "add 3 files"

 6. Отправить файл на внешний GitHub репозиторий.                             // git push

 7. Отредактировать содержание файла “new.txt” -
 написать информацию о себе (ФИО, возраст, количество домашних                // vim new.txt 

                                                                                 (ФИО)- Дроздов Павел Сергеевич
                                                                                 (age) 28
                                                                                 (Pets) 0

 8. Отправить изменения на внешний репозиторий.                               // git add // git commit -m "new 1 file"/
                                                                              // git push

 9. Создать файл preferences.txt                                              // touch preferences.txt

 10. В файл preferences.txt” добавить информацию о своих предпочтениях        // vim preferences.txt
